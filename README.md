# Healthcare Cost Drivers Analysis (In Progress)

## Project Overview  
This project explores key drivers of healthcare costs and utilization using a dataset of 100,000+ patient records. The goal is to identify patterns in lifestyle, demographics, and health conditions that impact medical expenses.

---

##  Business Questions  
- What factors contribute most to higher healthcare costs?  
- Do lifestyle habits (smoking, alcohol use) impact healthcare utilization?  
- Which demographic groups generate the highest total claims?  

---

##  Data Cleaning (Excel)  
- Reduced dataset from 54 columns to relevant features  
- Converted raw data into structured table format  
- Created separate sheet to preserve original dataset  

---

## Initial Insights (Early Findings)  
- Preliminary review suggests smokers tend to have higher total claims  
- Higher BMI appears to correlate with increased healthcare visits  

---

## Tools Used  
- Excel (data cleaning & initial analysis)  
- SQL (planned for deeper analysis)  
- Tableau (planned for visualization)
